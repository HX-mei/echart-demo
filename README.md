# echart-demo

## 项目技术栈
Vue全家桶：Vue + VueRouter + ElementUI + Echarts

## 安装依赖：
1. 网络请求`npm install echarts@4.9.0 --save`
2. Element`vue add element`

## VueCli工具版本
1. VueCli2.x

### Compiles and hot-reloads for development
